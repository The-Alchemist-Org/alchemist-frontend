# Usage

1. Import these:

```jsx
import {
  Banner, showBanner, BannerType, BannerProps,
} from './Banner';
```

2. Define a BannerMessage component:

```jsx
  const banner = (
    <Banner
      type={BannerType.INFO}
      title="A banner or toast"
      message="Served with a message"
    />
  );
```

3. Use function `showToast` to show toast:

```js
showToast(banner)
```
